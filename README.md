# Minecraft Configs

Pre-configured plugin configurations for Minecraft servers. Made by **KEYDAL Projects**.

Minecraft sunucuları için hazır plugin yapılandırmaları. **KEYDAL Projects** tarafından hazırlanmıştır.

---

## GrimAC - Gardiyan

Custom GrimAC anti-cheat configuration with full Turkish localization and KEYDAL branding.

GrimAC hile koruma sistemi için özelleştirilmiş yapılandırma.

### Features / Özellikler

- Full Turkish messages and punishment texts
- Custom gradient prefix: `[ɢᴀʀᴅɪʏᴀɴ]`
- Organized config with Turkish section headers
- Discord webhook integration
- Balanced detection thresholds
- Turkish punishment categories (HareketAnalizi, SavasKoruma, MenzilKoruma...)

### Installation / Kurulum

1. Install [GrimAC](https://modrinth.com/plugin/grimac) and [PacketEvents](https://modrinth.com/plugin/packetevents)
2. Copy all files from `grimac/` into `plugins/GrimAC/`
3. Restart your server
4. Configure `discord.yml` webhook URL if needed

### Compatibility / Uyumluluk

- GrimAC 2.3.74+
- Paper 1.20+
- PacketEvents 2.12+
